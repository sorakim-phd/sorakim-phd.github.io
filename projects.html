project 01

Enhancing SOP Accessibility in GMP Environments

accessibility

A human-centered approach to improving the usability of scanned SOP archives without altering validated documents.

information architecture document UX excel usability testing

What it addressed

  • Scanned SOP PDFs were not searchable or metadata-linked.
  • Analysts relied on manual scrolling and ambiguous document titles.
  • Retrieval delays increased cognitive fatigue during audits and urgent reviews.

Outcome

Built a metadata index for approximately 500 SOPs, improving retrieval speed from about 6–7 minutes to 1–2 minutes while maintaining full GMP compliance.

project 02

Human-Centered Review Interaction in Deviation & CAPA Management

interaction

A workflow design project focused on review clarity, feedback consistency, and turnaround time in validated documentation systems.

review interaction workflow design feedback analysis survey design

What it addressed

  • Different QA reviewers applied inconsistent feedback styles.
  • Redundant identifier entry produced friction during revisions.
  • Multi-day review loops emerged even for small clarifications.

Outcome

Developed a review interaction framework using Excel-based mapping, reviewer checklists, and latency heatmaps to improve feedback clarity while preserving validation integrity.

project 03

Human-Centered Cross-Reference System for GMP Documentation

traceability

A metadata-driven cross-reference system improving traceability and reducing cognitive load across deviation, CAPA, and related documentation records.

traceability metadata mapping python workflow visualization

What it addressed

  • Validated files could not be automatically interlinked.
  • Users manually cross-referenced IDs, status, and closure information.
  • Version inconsistency and lookup burden reduced inspection readiness.

Outcome

Built an Excel-based cross-reference matrix plus Python-supported analysis, achieving faster document tracking and lower version error risk without modifying validated systems.

project 04

Cognitive Pathways in Regulated Work Environments

cognition

A project mapping how analysts and reviewers actually think through procedural work, revealing where human reasoning diverges from formal workflow structure.

cognitive HCI field observation interviews network analysis

What it addressed

  • Procedural order did not always match actual reasoning order.
  • Users looped, revisited, and reinterpreted steps under pressure.
  • Cognitive bottlenecks emerged around terminology, cross-reference burden, and validation anxiety.

Outcome

Combined observation, interviews, Excel logging, and Python network visualization to identify non-linear reasoning patterns and translate them into human-centered design insight.

project 05

Workflow Feedback Dashboard for GMP Review Systems

analytics

A compliance-safe dashboard project visualizing reviewer workload, response times, and documentation bottlenecks using existing metadata.

workflow analytics dashboard excel python

What it addressed

  • Review cycles were opaque and often discussed only informally.
  • Turnaround delays and workload imbalance were hard to see early.
  • Teams lacked a shared visual layer for bottleneck detection.

Outcome

Created a dashboard from timestamps and reviewer metadata, enabling faster bottleneck identification and more transparent workflow monitoring without touching validated templates.

project 06

RCA Reasoning Scaffolder

decision support

A reasoning-support concept for root cause investigations, designed to help investigators explore hypotheses systematically instead of collapsing too early into single-cause explanations.

decision support reasoning design causal mapping investigation systems

What it addressed

  • RCA narratives often became linear and prematurely simplified.
  • Alternative hypotheses were difficult to sustain within existing document structures.
  • Investigators lacked an external reasoning layer to visualize causal relations.

Outcome

Designed a scaffold using investigation mapping, causal diagrams, and structured prompts to support broader hypothesis exploration while keeping validated systems unchanged.

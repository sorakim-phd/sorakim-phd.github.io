// paper_01
Accessibility, Document Usability, and Compliance-Constrained Design in Regulated Environments

Examines searchability, keyword pathways, and retrieval burden in scanned SOP archives, arguing that usability improvement can emerge through external support layers without modifying validated source files.

method: usability study (N=12) · corpus analysis
document UX accessibility
// paper_02
Reasoning Under Closure Pressure: Scaffolding Root Cause Analysis in Regulated Environments

Combines a pilot observation of premature causal convergence with an interface evaluation showing how scaffolds can preserve interpretive breadth under investigation pressure.

method: pilot observation (N=5) · scaffold evaluation (N=15)
reasoning investigation
// paper_03
Document Navigation as a Design Problem: Traceability and Behavioral Friction in GMP Environments

Combines cross-reference infrastructure with navigation tracking to reframe loopbacks and dependency gaps as signals of document-structure failure rather than individual error.

method: cross-reference evaluation (N=17) · navigation tracking (N=5)
navigation traceability
// paper_04
Workflow Friction and Observability in Deviation–CAPA Review Cycles

Reframes documentation review as an interaction system, examining how opacity in multi-stage quality workflows makes coordination burden and delay accumulation difficult to see.

method: prototype evaluation (N=6) · workflow analysis
workflow UX observability

Develops a broader HCI framework for regulated work environments, treating procedural documents and review systems as active interaction settings rather than neutral containers of compliance information.

method: conceptual synthesis · design framework
